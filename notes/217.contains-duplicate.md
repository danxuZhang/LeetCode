# 217.contains-duplicate

## Hashset

``` cpp
class Solution {
public:
    bool containsDuplicate(vector<int>& nums) {
        std::unordered_set<int> us;

        for (int num : nums) {
            if (us.count(num) != 0) 
                return true;
            us.insert(num);
        }

        return false;
    }
};
```
